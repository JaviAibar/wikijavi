## redondear un número

```python
# sin argumentos, redondea al entero más cercano
valor = round(0.5555)
print(valor) # 1

valor = round(0.4999999)
print(valor) # 0

valor = round(0.5555, 2) # el 2 es para los decimales
print(valor) # 0.56
```
