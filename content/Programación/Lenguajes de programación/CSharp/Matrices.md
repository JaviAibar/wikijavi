Existen 2 formas de hacer una matriz (rectangular - todos los vectores que la componen tienen el mismo tamaño - o escalonada - defines el tamaño de cada vector -)

rectangular:

- int[ , ] rect = new int [3 , 4];
    

escalonada:

- int[][] esc = new int[2][]; 
    
- esc[0] = new int[5];
    
- esc[1] = new int[2];
